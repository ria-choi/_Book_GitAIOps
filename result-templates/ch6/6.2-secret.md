# 6.2 Secret 관리 — 예상 결과

## 상태 확인

```
$ kubectl get secretproviderclass -n notiflex
NAME                AGE
valkey-secret       1m
```

## 체크리스트

- [ ] Secrets Store CSI Driver 설치됨
- [ ] Google Secret Manager에 Valkey 비밀번호 저장됨
- [ ] Workload Identity로 안전하게 접근 설정됨

## 이 단계 이후 아키텍처

```
[Google Secret Manager] → [CSI Driver] → [Pod Volume Mount]

GKE: notiflex-cluster
├── notiflex namespace
│   ├── Rollout: notiflex-api
│   ├── Valkey
```
