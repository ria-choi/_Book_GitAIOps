# 9.4 GitAIOps의 출현

## 실행 지침

이 프로젝트에서 Git, AI, Ops가 어떻게 연결되어 있는지 분석한다.

`JOURNEY.md`를 데이터 소스로 활용하여 **독자의 실제 여정** 기반으로 분석한다:

- **Git**: 코드, 매니페스트, JOURNEY.md(의사결정 기록)의 저장소
- **AI**: Claude Code가 코드 생성, 문제 해결, 아키텍처 결정을 보조 → JOURNEY.md에 기록
- **Ops**: JOURNEY.md의 도구 선택에 따른 GitOps 도구가 Git의 상태를 클러스터에 동기화

세 요소가 결합되어 "살아있는 운영 표준"이 어떻게 탄생하는지 설명한다.
JOURNEY.md 자체가 Git+AI+Ops의 산물임을 보여준다.
